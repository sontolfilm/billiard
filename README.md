# Billiard And Cafe

Website booking meja billiard dan pemesanan menu cafe berbasis PHP dan MySQL.

## Fitur

- Login admin dan user
- Booking meja billiard
- Kelola meja billiard
- Kelola menu makanan dan minuman
- Upload gambar menu
- Riwayat booking
- Dashboard admin

## Teknologi

- PHP
- MySQL / phpMyAdmin
- HTML
- CSS
- JavaScript
- XAMPP
